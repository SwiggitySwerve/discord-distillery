# Discord Distillery

Distill high-signal content from noisy Discord channels into actionable, section-by-section reports.

## Goal
Turn busy chat streams into a trustworthy digest of what actually matters:
- decisions
- action items
- blockers
- key insights
- useful resources
- open questions

Everything in reports should link back to source messages for auditability.

## Initial Docs
- `docs/scoring-rubric.md` — signal vs noise scoring framework
- `docs/report-schema.md` — canonical output schema (JSON + markdown)
- `docs/mvp-architecture.md` — MVP bot architecture and delivery plan

## MVP Principles
1. Aggressive noise filtering
2. Source-linked claims
3. Per-channel digesting
4. Deterministic structure, flexible language
